Part 1: Installing WireGuard

Overview

You’ll install WireGuard on both the Linux server (where the VPN will be hosted) and any client devices (Windows, macOS, etc.). Follow these steps to ensure everything is set up correctly before moving on to configuration.

Steps

  1. Check Your Kernel Version (Linux Server)
  2. WireGuard requires Linux kernel 5.6 or above, or backported modules on older distributions.
    bash uname -r

  3. If your distribution is older (e.g., Ubuntu 18.04), look up “WireGuard backports” to install the module.

  4. Update & Upgrade System Packages (Linux Server)

  5. Ensure your package index and installed packages are up-to-date:
    bash sudo apt update && sudo apt upgrade -y

  6. Install WireGuard (Linux Server)

  7. On Debian/Ubuntu:
    bash sudo apt install wireguard -y

  8. On CentOS/RHEL:
    bash sudo yum install epel-release -y sudo yum install kmod-wireguard wireguard-tools -y

  9. Check for Required Tools (Linux Server)

  10. Make sure the essential WireGuard tools (e.g., wg and wg-quick) are available:
    bash which wg && which wg-quick

  11. Verify WireGuard Installation (Linux Server)

  12. Confirm the version:
    bash wg --version

  13. You should see a version number and build date.

  14. Install WireGuard GUI or CLI (Windows)

  15. Download the installer from WireGuard.com and run the .exe file.

  16. Follow on-screen prompts and click Install.

  17. Install WireGuard App (macOS)

  18. Get the official WireGuard app from the Mac App Store.

  19. Drag the app to Applications if needed, then open it.

  20. Install WireGuard Mobile Apps (iOS/Android)

  21. On iOS, search WireGuard in the App Store.

  22. On Android, install WireGuard from the Google Play Store.

  23. Launch the WireGuard Client (All Platforms)

  24. Open the application (Linux: wg-quick, Windows: WireGuard GUI, macOS/iOS/Android: WireGuard app).

  25. You’ll configure it later in Part 3: Configuring Clients.

  26. (Optional) Enable Automatic Updates (Linux Server)

  27. Since security is crucial, consider enabling unattended upgrades or regularly run update checks: bash sudo apt install unattended-upgrades sudo dpkg-reconfigure unattended-upgrades


Alt-text: Terminal showing WireGuard installation commands.

Conclusion

WireGuard is now installed on your server and any client devices. In the next section, you’ll configure your Linux server with keys, network settings, and firewall rules to get your VPN fully operational.
